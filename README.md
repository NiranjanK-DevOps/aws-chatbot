# AWS Chatbot 

A conversational chatbot built using Amazon Lex V2. Configured intents, slots, responses, and tested the bot using the Lex console.

## Features

- Amazon Lex V2 chatbot
- Defined multiple intents and sample utterances
- Slot filling
- Error handling and fallback messages
- IAM permissions configured for secure access
- Tested using Lex Console

## Tech Stack

- Amazon Lex
- IAM
- AWS Console

## Description

This chatbot was created as part of a cloud learning project. It uses Lex to understand user input, match intents, request slot values, and generate responses.The screenshot of the developed chatbot has been uploaded in the repository.
